## Claim

Watch for the template being correctly updated (e.g. a 7-month term changed to a year) while the salary on file no longer matches what is actually paid — at scale, move to an HRIS with a file per employee.

## Source quote or paraphrase

Common failure: template updated correctly but salary no longer matches actual pay; at scale move to an HRIS with per-employee files.

## Why

When the documented terms diverge from reality, payroll, disputes, and DD all break against the wrong numbers; an HRIS keeps each employee's record true.
